# Tram Driver Safety — Attention Monitoring & Adaptive Assistance

## Overview
This repository supports PhD research investigating how driver attention 
and behavioural metrics can be dynamically integrated with environmental 
complexity to provide real-time assistance and prevent cognitive overload 
in urban tramway operations.

## Research Question
*How can driver attention and behavioural metrics be dynamically integrated 
with environmental complexity to provide real-time assistance to prevent 
cognitive overload?*

## Background
Urban tramway safety relies on driver vigilance in complex mixed-traffic 
environments. Cognitive load, fatigue, and distraction significantly 
increase accident risk. Current driver monitoring systems detect driver 
state but do not fully integrate the dynamic complexity of the surrounding 
environment.

## Research Framework
This work proposes a simulation-based framework that combines:
- **Non-intrusive behavioural indicators** — gaze patterns, fixations, 
  reaction times, braking behaviour
- **Environmental complexity modelling** — parameterised dynamic 
  interactions with pedestrians, cyclists, and vehicles
- **Adaptive assistance system design** — real-time context-aware 
  interventions targeted at cognitive overload prevention

## Methodology
- Simulation-based controlled experiments under varying environmental 
  complexity
- Vision-based, non-intrusive driver monitoring (gaze, posture, 
  fixation analysis)
- Temporal machine learning models for distraction and overload detection
- Integration of behavioural metrics with environmental context models

## Technologies
- Python (simulation, data analysis, ML models)
- OpenDRIVE / CARLA or equivalent simulation environment
- Computer vision libraries (OpenCV, MediaPipe)
- Machine learning frameworks (PyTorch / scikit-learn)
